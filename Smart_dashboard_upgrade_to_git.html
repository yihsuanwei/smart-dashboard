Smart Dashboard 升級指南

從舊版升級到 Git 版本，享受一鍵更新功能

預估時間：5-10 分鐘（大部分由 KIRO 自動完成）

📋升級流程總覽

你做 安裝 Git 下載並安裝 Git（只需做一次）
KIRO 做 其餘步驟 備份資料、下載新版、安裝環境、還原資料
升級後的好處
  • 一鍵更新 - 以後有新版本，雙擊 update.bat 就搞定
  • 不用再傳檔案 - 不用等人傳 zip、不用解壓縮覆蓋
  • 客戶資料不受影響 - 你的 uploaded_data 會保留
PART 1：你要做的事

📥安裝 Git（只需做一次）

Step 1：下載 Git

點擊下載：https://git-scm.com/download/win

選擇 「64-bit Git for Windows Setup」

Step 2：安裝

  1. 執行下載的安裝檔
  2. 全部按 Next 到底（不用改任何設定）
  3. 按 Install，等待完成
  4. 按 Finish

Step 3：確認安裝成功

  1. 按 Win + R，輸入 cmd，按 Enter
  2. 在黑色視窗輸入：git --version
  3. 看到版本號（如 git version 2.43.0）就是成功
安裝完成！ 接下來的步驟交給 KIRO 處理
PART 2：交給 KIRO

🤖讓 KIRO 完成升級

Git 安裝好之後，把下面整段文字複製貼給 KIRO：

請幫我把 Smart Dashboard 從舊版升級到 Git 版本。 【前提檢查】 1. 先執行 git --version 確認 Git 已安裝 2. 如果沒有安裝，請告訴我先手動安裝 Git：https://git-scm.com/download/win 【找到舊資料夾】 - 我的 Smart Dashboard 舊資料夾名稱類似 smart_dashboard_xxxxxx（有日期後綴） - 通常在桌面或文件資料夾 - 請幫我找到它的完整路徑 【升級步驟】（Git 已安裝的情況下執行） Step 1：備份客戶資料 - 把舊資料夾裡的 uploaded_data 資料夾複製到桌面，命名為 uploaded_data_backup Step 2：下載新版本 - 在桌面執行：git clone https://github.com/yihsuanwei/smart-dashboard.git - 這會建立一個新的 smart-dashboard 資料夾 Step 3：安裝 Python 環境 - 進入 smart-dashboard 資料夾 - 執行 setup.bat（用 call setup.bat 或直接執行） - 等待安裝完成 Step 4：還原客戶資料 - 把桌面的 uploaded_data_backup 資料夾 - 複製到 smart-dashboard 資料夾內 - 並重新命名為 uploaded_data（覆蓋原本的空資料夾） Step 5：測試啟動 - 執行 start.bat 確認可以正常啟動 - 看到 Dashboard 開啟就表示成功 【完成後告訴我】 1. 新的 Smart Dashboard 在哪裡（完整路徑） 2. 以後要用 Dashboard 就雙擊 start.bat 3. 收到更新通知就雙擊 update.bat 4. 舊的 smart_dashboard_xxxxxx 資料夾可以刪除了 5. 桌面的 uploaded_data_backup 也可以刪除了
KIRO 會幫你做什麼？
  • 檢查 Git 是否安裝
  • 找到你的舊 Smart Dashboard 資料夾
  • 備份你的客戶資料（uploaded_data）
  • 從 GitHub 下載最新版本
  • 安裝 Python 環境
  • 還原你的客戶資料
  • 測試確認可以正常使用

🎉升級完成後

想用 Dashboard 雙擊 start.bat
收到更新通知 雙擊 update.bat
就這樣！ 以後只要記得這兩個 bat 檔就好，超簡單

❓常見問題

Q：KIRO 說找不到 Git？

請確認 Git 安裝完成後，重新開啟 cmd 或 KIRO，再試一次。

Q：舊資料夾刪不掉？

請先關閉所有 Smart Dashboard 視窗和相關程式，再刪除。

Q：setup.bat 執行失敗？

可能是 Python 沒有安裝。請先安裝 Python 3.10+：https://www.python.org/downloads/

安裝時記得勾選「Add Python to PATH」

Q：升級後客戶資料不見了？

檢查桌面是否有 uploaded_data_backup，把它複製回 smart-dashboard 資料夾即可。
